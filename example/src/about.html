About Unify

Rebellion through simplicity.

Our Purpose

Unify is a modern, lightweight static site generator designed for frontend developers who want to build maintainable static sites with component-based architecture. Born from the need to restore serenity to web development in an era of endless framework churn and build complexity.

🌱 The fwdslsh Manifesto

Reclaim the DOM. Build pages with bits, not bundles.

In a world of endless frameworks and build complexity, we choose simplicity. We choose intention over automation, clarity over cleverness. No frameworks, no bloat, no compromise.

"A more peaceful future through purposeful technology."

Why We Created Unify

Frontend development has become unnecessarily complex. We believe in tools that empower without overwhelming, that build without breaking. Unify brings back the joy of building for the web with familiar HTML and Apache SSI syntax—no more copying and pasting headers, footers, and navigation across multiple pages.

Core Principles

🎯 Simplicity First

Zero learning curve with familiar Apache SSI syntax. No runtime complexity, no JavaScript frameworks—just pure HTML and CSS output.

🕊️ Peaceful Assembly

Your right to assemble pages bit-by-bit, calmly and predictably, without framework churn or build step dependencies.

⚡ Bun-Native Performance

Built exclusively for Bun with native APIs—HTMLRewriter, fs.watch, and Bun.serve—for maximum speed and efficiency.

🌿 Solarpunk Philosophy

Sustainable, purposeful technology choices. DIY-inspired design philosophy powered by joyful simplicity and digital minimalism.

🔒 Security-First

Path traversal prevention, input validation, and static output with no client-side template execution vulnerabilities.

🚀 Developer Experience

Live reload development server, incremental builds, smart dependency tracking, and comprehensive error messages.

How It Works

$ unify build
Processing: index.html → dist/index.html
Injecting includes into head…
Build complete in 112 ms

Run unify build --input src --output dist to generate your site. Includes are resolved recursively, head injections applied, and output written to dist/. Use unify serve for live preview and incremental rebuild.

Legacy of Thoughtful Aggregation

Dogpile pioneered the art of peaceful aggregation in web search—querying multiple engines and presenting clean, unified results from diverse sources. Unify reimagines that ethos for modern web development—assembling HTML fragments into coherent pages, but you remain in full control of every piece.

🔄 Then and Now

Dogpile (1996): Aggregated search results from multiple engines into one clean interface.

Unify (2025): Aggregates HTML fragments from multiple files into one cohesive site.

Both tools share the same philosophy: peaceful aggregation with purpose.
